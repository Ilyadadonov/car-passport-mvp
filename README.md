# 🚗 Car Passport — MVP

Кликабельный MVP-прототип мобильного приложения **Car Passport** — цифровой паспорт автомобиля.

## Демо

👉 **[Открыть прототип](https://<your-username>.github.io/car-passport-mvp/)**

## Что внутри

14 интерактивных экранов iOS-стиля:

| Экран | Описание |
|-------|----------|
| Онбординг | 3 слайда с ценностным предложением |
| Добавление авто | 2-шаговая форма + загрузка документов |
| Главная | Дашборд с карточкой авто, статусами, AI-ассистентом |
| Гараж | Мульти-авто с переключением (Kia K5 + BMW X3) |
| Профиль авто | Цифровой паспорт, VIN, документы |
| Деталь документа | ОСАГО/СТС/ПТС — номера, сроки, продление |
| QR-паспорт | QR-код для передачи истории при продаже |
| Расходы | Pie chart, категории, транзакции |
| Добавить расход | Форма с категориями, success state |
| История ТО | Timeline, AI-рекомендации, чеки |
| Добавить ТО | Форма записи обслуживания |
| Уведомления | Фильтры, приоритеты, empty state |
| Настройки | Тема, экспорт, AI, гараж |
| Подписка Pro | Paywall с тарифами и триалом |

## Ключевые фичи прототипа

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

1. Создайте репозиторий `car-passport-mvp` на GitHub
2. Замените `<your-username>` в README на свой username
3. Выполните:

```bash
git init
git add .
git commit -m "Car Passport MVP"
git branch -M main
git remote add origin https://github.com/<your-username>/car-passport-mvp.git
git push -u origin main
```

4. В настройках репозитория: **Settings → Pages → Source → GitHub Actions**
5. Деплой запустится автоматически. Через 1–2 минуты прототип будет доступен по адресу:

```
https://<your-username>.github.io/car-passport-mvp/
```
